# Official Python Documentation

You can also use python's official docs as follows:

- [wiki.python.org](https://wiki.python.org/moin/)

- [Python's own tutorial](https://docs.python.org/3.13/tutorial/index.html)
  
- [Refer Glossary](https://docs.python.org/3.13/glossary.html) to understand the terms and symbols used in the language (eg, garbage collection, generator, lambda func)

- [Refer Python Module Index](https://docs.python.org/3.13/py-modindex.html) to know more about any particulra built-in module 

- [Language Reference](https://docs.python.org/3.13/reference/index.html), mostly about the syntax and operators
